## 🎨 使用的CSS技术

### 1. **相对单位**
- 使用 `rem` 替代 `px` 作为单位
- 间距系统基于 CSS 变量：`--space-xs/sm/md/lg/xl`
- 字体大小使用响应式变量

### 2. **现代CSS函数**
- `clamp(min, preferred, max)` - 响应式字体和尺寸
- `min()` - 限制最大尺寸
- `max()` - 限制最小尺寸

### 3. **Flexbox 和 Grid**
- `flex: 1` - 自适应填充可用空间
- `flex-wrap: wrap` - 内容自动换行
- `grid-template-columns: repeat(auto-fit, minmax(...))` - 自适应网格
- `aspect-ratio` - 保持元素比例

### 4. **容器查询（预留）**
- 组件内部使用相对尺寸，支持未来容器查询
- 组件可独立适配其容器大小

### 5. **响应式断点**

```css
/* 平板及以下 */
@media (max-width: 768px) {
  /* 布局调整 */
}

/* 手机 */
@media (max-width: 480px) {
  /* 进一步优化 */
}
```

## 📐 尺寸转换对照

| 固定值 | 转换为 | 说明 |
|--------|--------|------|
| `width: 1440px` | `max-width: 90rem` + `width: 100%` | 最大宽度限制 |
| `height: 64px` | `min-height: 4rem` | 最小高度 |
| `width: 20%` | `flex: 0 0 20%` + `min/max-width` | Flex比例 |
| `padding: 80px 120px` | `clamp(2rem, 5vw, 5rem)` | 响应式间距 |
| `font-size: 48px` | `clamp(2rem, 5vw, 3rem)` | 响应式字体 |

## 🚀 优势

1. **无需刷新**: 窗口大小改变时实时响应
2. **更好的用户体验**: 在各种设备上都能舒适使用
3. **未来友好**: 支持折叠屏、平板电脑等新设备
4. **维护简单**: 使用CSS变量和现代技术，易于调整
5. **性能优化**: 纯CSS实现，无需JavaScript计算
